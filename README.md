# JavaScript algorithms

A colection of JavaScript Algorithms for all to enjoy. 

## Structure

Scripts are organized by topic:

- `arrays/` — Array and list operations and challenges
- `async/` — Promises and RxJS observables
- `crypto/` — Node.js crypto examples (encryption, HMAC, password hashing)
- `functional-programming/` — Closures, currying, first-class and high-order functions
- `js-fundamentals/` — JavaScript language features (iterators, symbols, maps, classes, etc.)
- `linked-lists/` — Linked list data structure and problems
- `math/` — Number algorithms (factorial, fibonacci, primes, fizzbuzz, etc.)
- `strings/` — String algorithms (palindrome, run-length encoding, word break, etc.)
- `typescript/` — TypeScript type-system features (generics, utility types, decorators, etc.)
- `html-css/` — HTML/CSS snippets
